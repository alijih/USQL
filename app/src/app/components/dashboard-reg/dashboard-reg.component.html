Welcome to your administration panel!

Here you can edit your website.

Home

* Here you can edit the information that appears in the main tab "home". Text "WHO WE ARE" and personal information.

Users

* Here you can view, add, edit and delete users and their permissions on the page.

* A simple logged-in user does not have access to this panel, but to the information that is hidden.

* An administrator can access all the functions of this panel.

* Remember that an Administrator user can delete other Administrator users.

Categories

* Here you can view, add and delete categories that will be displayed when adding reports.

* You can create a special category to incorporate hidden reports for all users where you can add reports only visible to registered and logged in users.

Reports

* The reports you add in the "LITERACY" category will be displayed in that tab.

* The reports you add in the "POLICY ANALISYS" category will be displayed in that tab.

* If you add hidden reports for all users, only registered users who are logged in will see it.

* The reports that you add with the active option "appear in home" will be shown in the following 3 reports of "latest report", if it is not active, they will be shown in "latest report" but when adding another report, it will no longer appear in the tab home.

* After adding a report with its main image (the one that is shown on the cover and in searches), you can enter to edit the report to add a secondary image that is displayed when viewing the report

Data Visuals

In the Visual Data options, you will be able to view, add and delete images that will be displayed in that tab.

Logout

Exit the administration panel
